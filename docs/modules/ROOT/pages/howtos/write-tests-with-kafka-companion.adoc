== Write tests with kafka companion

. If the `KafkaCompanion` is used through it's `ConsumerTask`s prepared in a `@BeforeEach`, use `TestInfo` parameter to generate better logs :

[source,java]
----
 @BeforeEach
void prepare(TestInfo testInfo) {
    log.infof("Preparing companion for %s", testInfo.getDisplayName());
    ...
----

. Prefere using usefull identifiers over generic ones:
[source,java]
----
//Nope
testService.createAndSaveField("field1", Type.STRING);
//Yes
testService.createAndSaveField("whenFieldAdded_eventSent", Type.STRING);
----
